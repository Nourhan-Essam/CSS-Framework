# 🚀 Modern Landing Page - Task 3

هذا المشروع عبارة عن صفحة هبوط (Landing Page) عصرية تم بناؤها كجزء من تدريب الـ Frontend، مع التركيز على تخصيص الـ UI باستخدام **Tailwind CSS**.

## ✨Features
- **Custom Branding:** تم تعديل الـ Theme الخاص بـ Tailwind لإضافة ألوان وهوية بصرية خاصة بالمشروع.
- **Fully Responsive:** التصميم متوافق تماماً مع جميع الشاشات (Mobile, Tablet, Desktop).
- **Interactive UI:** إضافة تأثيرات Hover وحركة ناعمة (Smooth Scroll) للتنقل بين الأقسام.
- **Modern Typography:** استخدام خط "Inter" لإعطاء مظهر احترافي ونظيف.

## 🛠️ الأدوات المستخدمة (Tech Stack)
- **HTML5:** للهيكل الأساسي.
- **Tailwind CSS:** لتنسيق الواجهة وبناء الـ Design System.
- **Lucide Icons:** للأيقونات التوضيحية.
- **Unsplash API:** للصور عالية الجودة.

## 🚀 How to run
1. قم بتحميل المشروع (Clone).
2. افتح ملف `index.html` في أي متصفح.
