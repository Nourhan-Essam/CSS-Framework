# 🚀 Modern Landing Page - Task 3 (CSS Frameworks)

A sleek, professional landing page built to demonstrate advanced UI customization using **Tailwind CSS**. This project focuses on cohesive design language, responsiveness, and clean code architecture.

## 🌐 Live Demo
Check out the live version here: 
👉 **[https://nourhan-essam.github.io/CSS-Framework/]**

---

## ✨ Project Objectives
- **Component Utilization:** Built using modular components like Navbar, Hero Section, and Features Grid.
- **Custom Branding:** Customized the default Tailwind theme with a unique color palette (`brandPrimary`, `brandSecondary`) and typography.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Accessibility:** Implemented semantic HTML tags for better screen reader support and SEO.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **Tailwind CSS:** Utility-first styling and custom theme configuration.
- **Lucide Icons:** Modern SVG icons.
- **Inter Font:** Clean, professional typography.
